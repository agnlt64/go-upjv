# Go UPJV

App de covoiturage pour l'UPJV t'as capté

## Quickstart

Systèmes Unix :
```bash
uv venv
source .venv/bin/activate
uv sync
uv run main.py
```

Windows :
```bash
uv venv
.venv\Scripts\activate
uv sync
uv run main.py
```

## Stack technique

- [uv](https://docs.astral.sh/uv/)
- Python 3.13
- Flask
- SQLAlchemy
- TailwindCSS
- Tout le reste est vanilla
